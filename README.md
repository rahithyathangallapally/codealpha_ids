# Smart Intrusion Detection System (IDS)

## Overview

The Smart Intrusion Detection System (IDS) is a Python-based cybersecurity tool developed to monitor network traffic and detect suspicious activities in real time.

The system captures packets, analyzes source and destination IP addresses, identifies communication protocols, and generates alerts when suspicious IP addresses are detected.

---

## Features

✅ Real-Time Packet Monitoring

✅ TCP, UDP Protocol Detection

✅ Suspicious IP Detection

✅ Alert Generation

✅ Packet Statistics

✅ Log File Creation

---

## Technologies Used

- Python
- Scapy
- Networking Concepts

---

## How It Works

1. Captures live network packets.
2. Extracts source and destination IP addresses.
3. Identifies packet protocol.
4. Compares source IPs against a suspicious IP database.
5. Generates alerts and stores them in a log file.
